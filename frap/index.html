Setting It Up
1
If your on a Windows on a Mac, you can just use Railsinstaller and bam! You've got ruby on rails with a bunch of other cool stuff.
2
Download the theme, stick it in the right directory and type 'bundle' in the terminal or command line to install the necessary gems.
3
Run 'jekyll', add or edit all the post you want, then upload all the contents in the _site folder to your live website. That's it.
